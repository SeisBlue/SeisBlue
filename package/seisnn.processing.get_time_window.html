seisnn.processing.get_time_window¶

seisnn.processing.get_time_window(anchor_time, station, trace_length=30, shift=0)[source]¶

Returns time window from anchor time.

Parameters

  • anchor_time – Anchor of the time window.

  • station (str) – Station name.

  • trace_length (int) – (Optional.) Trace length, default is 30.

  • or str shift (float) – (Optional.) Shift in sec, if ‘random’ will shift randomly within the trace length.

Return type

dict

Returns

Time window.
