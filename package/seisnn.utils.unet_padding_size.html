seisnn.utils.unet_padding_size¶

seisnn.utils.unet_padding_size(trace, pool_size=2, layers=4)[source]¶

Return left and right padding size for a given trace.

Parameters

  • trace (np.array) – Trace array.

  • pool_size (int) – (Optional.) Unet pool size, default is 2.

  • layers (int) – (Optional.) Unet stages, default is 4.

Returns

(left padding size, right padding size)
