# Building docs

Use [Sphinx](https://www.sphinx-doc.org/en/master/) to build your docs automatically.

## Install sphinx

    pip install Sphinx 
    
## Run sphinx-quickstart

Follow the link:

https://sphinx-rtd-tutorial.readthedocs.io/en/latest/sphinx-quickstart.html
